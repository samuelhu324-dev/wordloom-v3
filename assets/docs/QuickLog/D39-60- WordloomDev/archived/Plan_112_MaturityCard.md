🌈 视觉风格建议（与 Wordloom 当前 UI 兼容）
卡片：

8px 圆角

非强边框（#EAECEE 淡灰）

背景纯白

内部留白多一些（16~20px padding）

字体层级：

数字最大

标签次之

解释文本再小一点

颜色：

maturity 色带：蓝 → 绿 → 紫 → 灰（也可保持你当前配色）

Stable 阶段用更亮的绿色（增加“成品感”）
